## 1. About the Repository

<br />

This repository compiles the **fundamental programming tracks and core logic exercises** developed during the Full Stack Java Bootcamp at [Generation Brasil](https://brazil.generation.org/).

The scripts and packages contained here focus on the building blocks of software engineering using the Java language. It transitions from basic data output, conditional logic routing, and multi-layered repetition loops, up to runtime memory handling using arrays, advanced collections, and reusable custom methods.

<br />

## 🛠️ Concepts & Skills Covered
- **Java Fundamentals**: Variables, operators, and control structures.
- **Object-Oriented Programming (OOP)**: Implementation of classes, inheritance, polymorphism, and encapsulation.
- **Collections**: Managing data efficiently using Lists, Sets, and Maps.
- **Problem Solving**: Logic exercises designed to build a robust backend foundation.
